# 目標
まずは、１つのハンコでできるようにし、その後、複数個のハンコをそれぞれ認識するようにしたい。

複数のハンコの場合、ハンコの上の部分の形（もしくは色）をそれぞれ変えて、その形（または色）で認識し掴む。
